Data Requirements for RAG System

1. Purpose

This document outlines the basic data needed for our Retrieval-Augmented Generation (RAG) System to answer student questions using course materials.

2. Required Data

  • Lecture Notes
  • Study Guides
  • Course Handbooks
  • Textbooks (if permitted by copyright)
  • Any additional materials provided by instructors

3. File Format

  • PDF only
  • Documents should be clear and readable (not handwritten or scanned images).
  • Text should be selectable (not just images of text).

4. Data Processing Considerations

  • Documents will be indexed for search and retrieval.
  • The system will answer questions only based on the uploaded PDFs.
  • If a document is updated, it should be re-uploaded to keep answers accurate.

5. Next Steps

  • Gather and upload relevant PDF materials.
  • Ensure all documents contain useful, structured information.
  • Start testing the system with sample questions.
